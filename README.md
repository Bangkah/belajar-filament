## Belajar Filament

Project ini dibuat sebagai latihan mengikuti seri tutorial Laravel Filament untuk membangun dashboard administrasi modern berbasis Laravel. Di dalam project ini, kita mempelajari cara membuat struktur resource yang rapi dan modular, mulai dari pembuatan model, migration, konfigurasi Filament, hingga pembuatan fitur CRUD (Create, Read, Update, Delete) pada data Category.

Tujuannya adalah agar saya dapat memahami bagaimana Filament bekerja pada Laravel, serta mampu mengembangkan panel admin yang bersih, terstruktur, dan mudah dikembangkan untuk kebutuhan aplikasi yang lebih besar di masa mendatang.

### Yang Dipelajari dalam Project Ini:
- Membuat project Laravel baru
- Menginstal dan mengkonfigurasi Filament Admin Panel
- Membuat model dan migration (Category)
- Membuat CRUD Category dengan Resource
- Memisahkan konfigurasi Form dan Table ke dalam folder Schemas dan Tables
- Memahami struktur halaman Filament (List, Create, Edit)
